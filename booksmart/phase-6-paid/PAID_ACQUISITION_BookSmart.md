## 3. TRACKING & MEASUREMENT

### 3.1 UTM Convention

```
URL Structure:
  {base_url}?utm_source={source}&utm_medium={medium}&utm_campaign={campaign}&utm_content={content}&utm_term={keyword}

Values:
  source: google | meta | linkedin | email
  medium: cpc | social | email
  campaign: brand_v1 | competitor_v1 | problem_solution_v1 | meta_lal_v1 | linkedin_demo_v1
  content: ad_group_name (snake_case)
  term: keyword_id or audience_id
```

**Example:**
```
https://booksmart.io/demo?utm_source=google&utm_medium=cpc&utm_campaign=problem_solution_v1&utm_content=no_show&utm_term=dental+no-show+solution
```

### 3.2 Conversion Tracking Setup

| Platform | Event | Setup Method |
|---|---|---|
| **Google Ads** | Lead (form submit), Phone Call, Demo Request | Google Tag via GTM + Google forwarding number |
| **Meta Pixel** | PageView, ViewContent, Lead, Schedule (booking demo) | Pixel base + CAPI (see 3.3) |
| **LinkedIn Insight** | Lead, PageVisit (landing page) | Insight Tag via GTM |

**Key conversion actions:**
1. `Lead` — Form submission (any page)
2. `Demo Request` — Booked demo on calendar (higher-value)
3. `Phone Call` — Call connected >30s
4. `Signup` — Account created (post-demo)

### 3.3 Meta Conversions API (CAPI)

**Required fields sent server-side (via backend or GTM server container):**
```
event_name: Lead, DemoRequest, Signup
event_time: unix_timestamp
event_source_url: page_url_with_utm
user_data:
  em: sha256(email)
  ph: sha256(phone)
  fn: sha256(first_name)
  ln: sha256(last_name)
  zp: sha256(zip)
  client_ip_address
  client_user_agent
event_id: unique_id (for dedup)
action_source: website
```

**Dedup logic:** Match on `event_id` between browser pixel and CAPI. If both arrive, server event wins (kept), browser event deduped.

### 3.4 Dashboard Metrics & Reporting

| Metric | Definition | Target |
|---|---|---|
| **CPA** | Cost per lead (form submit) | < $50 |
| **CPL (Demo)** | Cost per booked demo | < $80 |
| **Conversion Rate (CVR)** | Lead rate from landing page visitors | > 8% |
| **ROAS** | Revenue attributed / ad spend (view-through) | > 3x |
| **CTR** | Click-through rate (Google) | > 3% |
| **No-Show Lead Quality** | % of leads that are dental practices | > 70% |

**Tool:** Google Looker Studio (free) — connect Google Ads, Meta, LinkedIn via Supermetrics or manual CSV pulls. Auto-refresh weekly.

### 3.5 Attribution Window

| Channel | Click Window | View-Through Window |
|---|---|---|
| Google Ads | 30-day click | None |
| Meta | 7-day click | 1-day view |
| LinkedIn | 7-day click | 1-day view |
| Email | 7-day click | None |

**Note:** View-through conversions are tracked but flagged separately. Primary optimization uses click-through attribution only.
